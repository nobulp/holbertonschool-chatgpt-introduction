# holbertonschool-chatgpt-introduction

## Description
This project is an introduction to using ChatGPT as a tool to improve code quality and efficiency.  
It focuses on identifying common bugs, fixing them, and adding basic input validation and logic (win/draw conditions, etc.).

## Learning Objectives
- Identify errors in given code samples
- Fix bugs and edge cases
- Add input validation to prevent crashes
- Improve code readability with small refactors
- Test programs with different user inputs
